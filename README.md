# Brain Training TOEFL Listening Part B

Website tugas kuliah berbasis **Next.js + Tailwind CSS** dengan 3 mini game:

1. **Landing page** dengan input nama tanpa login
2. **TOEFL Memory** berbasis Google Gemini API
3. **Stroop Focus** untuk latihan fokus warna
4. **Sequence** untuk melatih memori urutan kata
5. **Leaderboard** berbasis Upstash Redis di Vercel dengan TTL 1 jam
